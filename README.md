# 🏋️‍♂️ Aplicativo de Avaliação Física

Este é um projeto desenvolvido em **Python** focado em automatizar o processo de avaliação física para Personal Trainers. O objetivo é facilitar o cálculo de composição corporal e a geração de relatórios precisos para os alunos.

## 🚀 Funcionalidades

- **Cálculo de Composição Corporal:** Implementação de protocolos como Pollock (7 dobras).
- **Classificação por Faixa Etária:** Lógica personalizada para classificar o % de gordura.
- **Interface Intuitiva:** Desenvolvimento focado em usabilidade para o dia a dia na academia.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python 3.x
- **Versionamento:** Git & GitHub
- **Ambiente:** VS Code

## 📈 Contexto do Projeto

Este software nasceu da necessidade real de otimizar meu trabalho como Personal Trainer na **Academia Vibe Fit**. Ele faz parte da minha jornada de transição de carreira para a tecnologia, unindo meus conhecimentos em Educação Física com os estudos em **Análise e Desenvolvimento de Sistemas (ADS)**.
